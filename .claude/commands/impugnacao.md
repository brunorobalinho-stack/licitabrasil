# /impugnacao - Analise de Edital

Edital: **$ARGUMENTS**

## Protocolo:
1. Ler edital completo
2. Identificar clausulas restritivas
3. Verificar conformidade Lei 14.133/21
4. Comparar valores com mercado
5. Identificar vicios formais
6. Avaliar viabilidade (alta/media/baixa)

## Output:
- Irregularidades + artigo violado
- Recomendacao: impugnar ou nao
- Se sim: rascunho dos argumentos
- Prazo e proximos passos
